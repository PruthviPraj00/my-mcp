# Tailwind CSS Skeleton - FlyonUI

h3:
  - FlyonUI

p:
  - components

p:
  - The skeleton component is utilized to display the loading state of a component.

table:
| Class Name | Type | Description |
| --- | --- | --- |
| skeleton | Component | Displays a markup’s structure. |
| skeleton-animated | Modifier | Displays skeleton with loading animation. |

h2:
  - Variants

h3:
  - Default skeleton

p:
  - Apply theskeletoncomponent class to any markup to showcase it’s structure.

Code:
<div :dir="isRtl ? 'rtl' : 'ltr'" class="overflow-hidden" id="previewFrame" x-ref="previewFrame">
 <div class="bg-base-200/20 border-neutral/10 rounded-box not-prose flex w-full flex-[1_0_0] flex-wrap gap-3 border p-3 sm:p-6">
  <div class="skeleton h-32 w-32">
  </div>
 </div>
</div>

h3:
  - Animated skeleton

p:
  - Apply theskeleton-animatedmodifier class to any markup to showcase it’s structure with loading animation.

Code:
<div :dir="isRtl ? 'rtl' : 'ltr'" class="overflow-hidden" id="previewFrame" x-ref="previewFrame">
 <div class="bg-base-200/20 border-neutral/10 rounded-box not-prose flex w-full flex-[1_0_0] flex-wrap gap-3 border p-3 sm:p-6">
  <div class="skeleton skeleton-animated h-32 w-32">
  </div>
 </div>
</div>

h3:
  - Active skeleton

p:
  - Add theanimate-pulseanimation class to any markup to display its structure with an active animation. For additional options, refer to the TailwindCSSanimationdocumentation.

Code:
<div :dir="isRtl ? 'rtl' : 'ltr'" class="overflow-hidden" id="previewFrame" x-ref="previewFrame">
 <div class="bg-base-200/20 border-neutral/10 rounded-box not-prose flex w-full flex-[1_0_0] flex-wrap gap-3 border p-3 sm:p-6">
  <div class="skeleton h-32 w-32 animate-pulse">
  </div>
 </div>
</div>
