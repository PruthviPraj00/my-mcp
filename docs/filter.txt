p:
  - A group of radio buttons that lets you pick one option, hiding the rest, with a reset button to change your choice.

table:
| Class Name | Type | Description |
| --- | --- | --- |
| filter | Component | Applies toformsordivscontaining radio buttons for item filtering. |
| filter-reset | Part | Reset button alternative for non-form elements. |

h3:
  - Filter with HTML form: radio buttons and reset button

p:
  - A HTML form for filtering with radio buttons and a reset option.

```html
<form class="filter">
 <input class="btn btn-square" type="reset" value="Ã"/>
 <input aria-label="Tea" class="btn" name="drink" type="radio"/>
 <input aria-label="Coffee" class="btn" name="drink" type="radio"/>
 <input aria-label="Smoothie" class="btn" name="drink" type="radio"/>
</form>

```
h3:
  - Implementing filter without HTML forms

p:
  - Use this method when HTML forms are not an option.

```html
<div class="filter">
 <input aria-label="All" class="btn filter-reset" name="destination" type="radio"/>
 <input aria-label="Jungle" class="btn" name="destination" type="radio"/>
 <input aria-label="Beach" class="btn" name="destination" type="radio"/>
 <input aria-label="Mountain" class="btn" name="destination" type="radio"/>
</div>

```
