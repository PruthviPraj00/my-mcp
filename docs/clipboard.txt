h2:
  - Basic example

h3:
  - Default

p:
  - To copy text from a specific element, target it using itsIdand assign it to thedata-clipboard-targetattribute in the trigger button. Additionally, set the value of thedata-clipboard-actionattribute tocopy. This setup ensures that clicking the button will copy the text from the targeted element to the clipboard.

p:
  - The following example demonstrates the default usage of the copy clipboard component.

```html
<div class="rounded-box inline-flex items-center gap-1 p-1 shadow-base-300/20 shadow-sm">
 <code class="px-2 text-base-content text-sm font-medium" id="clipboard-basic">
  npm install flyonui
 </code>
 <button aria-label="Copy text to clipboard" class="js-clipboard btn btn-square btn-text" data-clipboard-action="copy" data-clipboard-target="#clipboard-basic" type="button">
  <span class="js-clipboard-default icon-[tabler--clipboard] size-5 transition">
  </span>
  <span class="js-clipboard-success icon-[tabler--clipboard-check] text-primary hidden size-5">
  </span>
 </button>
</div>

```

h3:
  - Copy source code block

p:
  - The example below illustrates the use of the copy clipboard component in source code block.

```html
<div class="mockup-code bg-base-200/40 rounded-box relative pb-0 before:content-none">
 <pre class="overflow-x-auto text-sm" id="clipboard-source-code"><code>const clipboard = new ClipboardJS(el, {
    text: trigger => {
    const clipboardText = trigger.dataset.clipboardText
    if (clipboardText) return clipboardText
    const clipboardTarget = trigger.dataset.clipboardTarget
    const $element = document.querySelector(clipboardTarget)
    if ($element.tagName === 'SELECT' || $element.tagName === 'INPUT' || $element.tagName === 'TEXTAREA')
      return $element.value
    else return $element.textContent
    }
})</code></pre>
 <button aria-label="Copy text to clipboard" class="js-clipboard tooltip absolute end-2 top-2" data-clipboard-action="copy" data-clipboard-success-text="Copied!" data-clipboard-target="#clipboard-source-code" type="button">
```
