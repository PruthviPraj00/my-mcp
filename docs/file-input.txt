# Tailwind CSS File input - FlyonUI

h3:
  - FlyonUI

p:
  - forms

p:
  - File inputs enable users to choose files from their device, allowing them to upload documents, images, or other types of files.

table:
| Class Name | Type | Description |
| --- | --- | --- |
| input | Component | Base class for form input. |
| input-floating | Component | Base class for floating form input. |
| label-text | Part | Base class for title label text. |
| input-floating-label | Part | Floats the label to the top. |
| helper-text | Part | Base class for helper label text. |
| is-valid | State | Adds success style to the input. |
| is-invalid | State | Adds error style to the input. |
| file:{tw-utility-class} | Modifier | Adds tailwind classes to update file input. |
| input-xs | Size | Input with Extra small size. |
| input-sm | Size | Input with Smaller size. |
| input-md | Size | Input with medium(default) size. |
| input-lg | Size | Input with Larger size. |
| input-xl | Size | Input with Extra large size. |

h2:
  - Basic example

h3:
  - Default

p:
  - Basic file input example.

```html
<input aria-label="file-input" class="input max-w-sm" type="file"/>

```
h3:
  - Label

p:
  - Basic file input example with label.

```html
<div class="max-w-sm">
 <label class="label-text" for="fileInputLabel">
  Pick a file
 </label>
 <input class="input" id="fileInputLabel" type="file"/>
</div>

```
h2:
  - Size

h3:
  - File input sizes

p:
  - Add responsive classinput-{size}where{size} = xs|sm|md|lg|xlfor file input of different sizes.

```html
<input aria-label="file-input" class="input max-w-sm input-xs" type="file"/>
<input aria-label="file-input" class="input max-w-sm input-sm" type="file"/>
<input aria-label="file-input" class="input max-w-sm" type="file"/>
<input aria-label="file-input" class="input max-w-sm input-lg" type="file"/>
<input aria-label="file-input" class="input max-w-sm input-xl" type="file"/>

```

h2:

h3:
  - File input with button

p:
  - Thefile:prefix in Tailwind allows us to style the::file-selector-button, and it can be combined with thebtnclass.

```html
<input aria-label="file-input" class="block text-sm file:uppercase file:btn file:btn-primary file:me-3" type="file"/>

```
h3:
  - Disabled

p:
  - Add thedisabledboolean attribute on an file input to remove pointer events, and prevent focusing.

```html
<input aria-label="file-input" class="input max-w-sm" disabled="" type="file"/>

```
h3:

h3:
  - Multiple files

p:
  - Addmultipleattribute to select more then 1 file.

```html
<input aria-label="file-input" class="input max-w-sm" multiple="" type="file"/>

```
