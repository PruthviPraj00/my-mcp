# Tailwind CSS Radial progress - FlyonUI

h3:
  - FlyonUI

p:
  - components

p:
  - Radial progress can visually represent either the completion of a task or the elapsed passage of time.

table:
| Class Name | Type | Description |
| --- | --- | --- |
| radial-progress | Component | Base class for radial progress component. |

p:
  - The radial progress component requires the--valueCSS variable to define its progress percentage.

p:
  - To adjust its size, use the--sizeCSS variable, which defaults to 5rem.

p:
  - To modify the thickness of the progress ring, use the--thicknessCSS variable, which is set to 10% of the size by default.

p:
  - For Radial progress we need to use adivinstead of theprogresstag because browsers canât show text insideprogresstag, and Firefox doesnât render pseudo-elements insideprogresstag at all.

p:
  - Addingrole="progressbar"makes it accessible to screen readers as well.

h2:
  - Variants

h3:
  - Default

p:
  - Utilize theradial-progresscomponent class and adjust the progress value by setting the CSS variable--value:*in
the style attribute of the radial progress component, where*represents a numerical value.

```html
<div aria-label="Radial Progress" class="radial-progress" role="progressbar" style="--value:75">
</div>

```

h2:
  - Sizes

h3:
  - Custom size & thickness

p:
  - Modify the size and thickness of the progress by defining the CSS variables--size:*and--thickness:*within the style attribute of the radial progress component, where*denotes a numerical value along with the unit.

```html
<div aria-label="Radial Progressbar" class="radial-progress" role="progressbar" style="--value:70;--size:12rem;--thickness:2px">
 70%
</div>
<div aria-label="Radial Progressbar" class="radial-progress" role="progressbar" style="--value:70;--size:12rem;--thickness:2rem">
 70%
</div>
<div aria-label="Radial Progressbar" class="radial-progress bg-primary/10 text-primary border-4 border-transparent" role="progressbar" style="--value:70;--size:12rem;--thickness:1rem">
 70%
</div>

```
