# Tailwind CSS Link - FlyonUI

h3:
  - FlyonUI

p:
  - content

p:
  - The link component facilitates setting hyperlinks from inline text items, buttons, or cards to navigate between pages or external websites.

table:
| Class Name | Type | Description |
| --- | --- | --- |
| link | Component | Adds underline to a text. |
| link-hover | Style | Only show underline on hover. |
| link-animated | Style | Show underline on hover with animation. |
| link-neutral | Color | Applies’neutral’ color to link. |
| link-primary | Color | Applies’primary’ color to link. |
| link-secondary | Color | Applies’secondary’ color to link. |
| link-accent | Color | Applies’accent’ color to link. |
| link-success | Color | Applies’success’ color to link. |
| link-info | Color | Applies’info’ color to link. |
| link-warning | Color | Applies’warning’ color to link. |
| link-error | Color | Applies’error’ color to link. |

h2:
  - Variants

h3:
  - Default

p:
  - Thelinkcomponent class serves as the standard class for the basic anchor tag.

```html
<a class="link" href="#">
 Default link
</a>

```
h3:
  - Underline on hover

p:
  - Apply thelink-hovermodifier class to make links underlined when hovered over.

```html
<a class="link link-hover" href="#">
 Underline when hovered over.
</a>

```
h3:
  - Animated underline

p:
  - Apply thelink-animatedmodifier class to make links underlined with animation on hovered over.

```html
<a class="link link-animated" href="#">
 Animated underline when hovered over.
</a>

```
h2:
  - Colors

h3:
  - Semantic links

p:
  - Utilize thelink-{semantic-color}modifier class to assign specific semantic colors to links.

```html
<a class="link link-neutral" href="#">
 Neutral link
</a>
<a class="link link-primary" href="#">
 Primary link
</a>
<a class="link link-secondary" href="#">
 Secondary link
</a>
<a class="link link-accent" href="#">
 Accent link
</a>
<a class="link link-info" href="#">
 Info link
</a>
<a class="link link-success" href="#">
 Success link
</a>
<a class="link link-warning" href="#">
 Warning link
</a>
<a class="link link-error" href="#">
 Error link
</a>

```
h3:
  - Custom Color

p:
  - Use the--link-colorfor the custom color of the link.

```html
<a class="link [--link-color:purple]" href="#">
 Default
</a>
<a class="link link-hover [--link-color:purple]" href="#">
 Underline on Hover
</a>
<a class="link link-animated [--link-color:purple]" href="#">
 Animated Underline
</a>

```
h2:
  - Illustrations

h3:
  - In paragraphs

p:
  - Below is an example demonstrating how to use links within paragraph.

```html
<p>
 Tailwind CSS resets the appearance of links by default.
 <br/>
 To restore them to a standard look, add the 'link' class, making it appear as a
 <a class="link" href="#">
  normal link
 </a>
 once more.
</p>

```
h3:
  - In cards

p:
  - Below is an example demonstrating how to use links in card.

```html
<div class="card max-w-sm">
 <div class="card-body">
  <h5 class="card-title mb-0">
   Card title
  </h5>
  <div class="text-base-content/50 mb-3">
   Card subtitle
  </div>
  <p class="mb-4">
   Some quick example text to build on the card title and make up the bulk of the card's content.
  </p>
  <div class="card-actions">
   <a class="link link-primary link-hover" href="#">
    Card link
   </a>
   <a class="link link-primary link-animated" href="#">
    Another link
   </a>
  </div>
 </div>
</div>

```

  policy

 .
