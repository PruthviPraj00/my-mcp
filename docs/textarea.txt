# Tailwind CSS Textarea - FlyonUI

h3:
  - FlyonUI

p:
  - forms

p:
  - A textarea is a multi-line text input field that allows users to enter longer text.

table:
| Class Name | Type | Description |
| --- | --- | --- |
| textarea | Component | Basic textarea field. |
| textarea-floating | Component | Floating textarea field. |
| textarea-floating-label | Part | Floats the label to the top. |
| label-text | Part | For label. |
| helper-text | Part | For helper text. |
| is-valid | State | Adds success style to the input. |
| is-invalid | State | Adds error style to the input. |
| textarea-xs | Size | Extra small Textarea size. |
| textarea-sm | Size | Smaller Textarea size. |
| textarea-md | Size | Medium(default) Textarea size. |
| textarea-lg | Size | Larger Textarea size. |
| textarea-xl | Size | Extra large Textarea size. |

h2:
  - Basic example

h3:
  - Default

p:
  - Basic textarea example.

```html
<textarea aria-label="Textarea" class="textarea max-w-sm"></textarea>

```
h3:
  - Label and placeholder

p:
  - Basic textarea with placeholder.

```html
<div class="sm:w-96">
 <label class="label-text" for="textareaLabel">
  Your bio
 </label>
 <textarea class="textarea" id="textareaLabel" placeholder="Hello!!!"></textarea>
</div>

```
h2:
  - Types

h3:
  - Default

```html
<textarea class="textarea max-w-sm" placeholder="Hello!!!"></textarea>

```
h3:
  - Floating label

p:
  - Thetextarea-floatingstyle is used to define the appearance of the textarea, while thetextarea-floating-labelis the container for the label text.

```html
<div class="textarea-floating sm:w-96">
 <textarea class="textarea" id="textareaFloating" placeholder="Hello!!!"></textarea>
 <label class="textarea-floating-label" for="textareaFloating">
  Your bio
 </label>
</div>

```

h3:

<textarea class="textarea textarea-sm max-w-sm" placeholder="Hello!!!"></textarea>
<textarea class="textarea max-w-sm" placeholder="Hello!!!"></textarea>

h3:
  - Disable

p:
  - Add thedisabledboolean attribute on an textarea to remove pointer events, and prevent focusing.

```html
<div class="w-full sm:w-96">
 <label class="label-text sr-only" for="textareaDisabledefault">
  Your bio
 </label>
 <textarea class="textarea" disabled="" id="textareaDisabledefault" placeholder="Hello!!!"></textarea>
</div>
<div class="textarea-floating sm:w-96">
 <textarea class="textarea" disabled="" id="textareaDisabledoating" placeholder="Hello!!!"></textarea>
 <label class="textarea-floating-label" for="textareaDisabledoating">
  Your bio
 </label>
</div>

```
h3:
  - Readonly

p:
  - Add thereadonlyboolean attribute on an textarea to prevent modification of the textarea value.

```html
<div class="w-full sm:w-96">
 <label class="label-text sr-only" for="textareaReadonly">
  Your bio
 </label>
 <textarea class="textarea" id="textareaReadonly" placeholder="Hello!!!" readonly=""></textarea>
</div>

```
