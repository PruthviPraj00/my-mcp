h3:

p:
  - Basic date picker example.

```html
<input class="input max-w-sm" id="flatpickr-date" placeholder="YYYY-MM-DD" type="text"/>

```

h3:
  - Floating

p:
  - Floating label example.

```html
<div class="input-floating max-w-sm">
 <input class="input" id="flatpickr-floating" placeholder="YYYY-MM-DD" type="text"/>
 <label class="input-floating-label" for="flatpickr-floating">
  Date
 </label>
</div>

```
h2:
  - Validation states

h3:
  - Success state

p:
  - Success state can be show usingis-validclass.

```html
<div class="max-w-sm">
 <label class="label-text" for="flatpickrStateSuccess">
  Date
 </label>
 <input class="input is-valid flatpickr-success" id="flatpickrStateSuccess" placeholder="YYYY-MM-DD" type="text"/>
 <span class="helper-text">
  Helper text
 </span>
</div>
<div class="input-floating max-w-sm">
 <input class="input is-valid flatpickr-success" id="flatpickrFloatingStateSuccess" placeholder="YYYY-MM-DD" type="text"/>
 <label class="input-floating-label" for="flatpickrFloatingStateSuccess">
  Date
 </label>
 <span class="helper-text ps-3">
  Helper text
 </span>
</div>

```
h3:
  - Error state

p:
  - Error state can be show usingis-invalidclass.

```html
<div class="max-w-sm">
 <label class="label-text" for="flatpickrStateError">
  Date
 </label>
 <input class="input is-invalid flatpickr-error" id="flatpickrStateError" placeholder="John Doe" type="text"/>
 <span class="helper-text">
  Helper text
 </span>
</div>
<div class="input-floating max-w-sm">
 <input class="input is-invalid flatpickr-error" id="flatpickrFloatingStateError" placeholder="John Doe" type="text"/>
 <label class="input-floating-label" for="flatpickrFloatingStateError">
  Date
 </label>
 <span class="helper-text ps-3">
  Helper text
 </span>
</div>

```
